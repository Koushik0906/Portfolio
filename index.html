A comprehensive Java-based web application that simulates an online railway reservation platform with user and admin modules.

Key Features:

  • User Module: Registration, login, train search, ticket booking, booking history, and profile management
  • Admin Module: Train management (add, update, delete), view all trains, and profile management
  • Database: Oracle SQL with JDBC for data operations
Java JSP Servlets HTML/CSS Bootstrap Oracle SQL JDBC Apache Maven Apache Tomcat

2. Age Calculator

A simple web application that computes a person's exact age based on their date of birth, displaying years, months, and days elapsed.

Key Features:

  • Date of birth input via form field
  • Real-time age calculation (years, months, days)
  • Responsive design for desktop and mobile
  • Client-side execution with no backend required
HTML CSS JavaScript

3. BMI Calculator

A lightweight web application designed to calculate Body Mass Index (BMI) quickly and display health categories.

Key Features:

  • User inputs weight and height
  • Automatic BMI calculation with formula: BMI = weight / (height²)
  • Display of result with category label
  • Browser-based execution, works offline
HTML CSS JavaScript
